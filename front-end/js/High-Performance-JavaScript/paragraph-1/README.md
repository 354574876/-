## 脚本位置
请记住，浏览器在解析到body标签之前，不会渲染页面的任何部分。把脚本放在页面顶部会导致明显的延迟，通常表现为显示空白页面，用户无法浏览内容，页无法页页面进行交互。  
即使现在很多浏览器支持并行下载Javascript文件，但是页面荏苒必须等待所有javascript代码下载并执行完成才能继续。  
所以应将所有script标签放在body标签地步
